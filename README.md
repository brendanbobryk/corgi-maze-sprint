A fun maze navigation game built with React, where you guide a corgi through a randomly generated maze to reach the goal!  
This project is part of a daily game challenge focused on building small, complete games to practice React fundamentals.

## Features

- Randomly generated maze each round  
- Guaranteed solvable paths from start to goal  
- Arrow key movement (up, down, left, right)  
- Timer tracking how long it takes to reach the goal  
- Restart button for new mazes  
- Polished dark-mode UI for easy viewing  
- Cute corgi and goal emojis ğŸ¶ğŸ 

## Technologies Used

- React (with hooks)
- JavaScript
- CSS

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/brendanbobryk/corgi-maze-sprint.git

2. Navigate to the project folder:
    ```bash
    cd corgi-maze-sprint

3. Install dependencies:
    ```bash
    npm install

4. Start the development server:
    ```bash
    npm run dev

5. Open your browser at the local address shown in the terminal (usually http://localhost:5173) to play!
